# UI Tokens â€“ Quick Reference

## Base
| Token  | Usage |
|--------|-------|
| **bg** | Main page/app background. |
| **fg** | Default text and icon color on `bg`. |

## Surfaces
| Token         | Usage |
|---------------|-------|
| **surface-1** | Cards, panels, input backgrounds. |
| **surface-2** | Popovers, menus, raised buttons. |
| **surface-3** | Modals, top layers. |
| **muted**     | Low-emphasis fills (list rows, subtle chips). |
| **overlay**   | Page dimmer behind modals/drawers. |

## Text
| Token          | Usage |
|----------------|-------|
| **text**       | Standard text on neutral surfaces. |
| **text-muted** | Secondary text (placeholders, hints). |
| **text-inverse** | Text on dark/colored backgrounds. |

## Structure
| Token     | Usage |
|-----------|-------|
| **border** | Component outlines (inputs, cards). |
| **divider** | Stronger section/item separators. |

## Brand & Accent
| Token          | Usage |
|----------------|-------|
| **primary** / **onPrimary**     | Primary actions, brand surfaces / text on them. |
| **secondary** / **onSecondary** | Secondary actions, accents / text on them. |
| **link**                          | Inline links. |

## States
| Token          | Usage |
|----------------|-------|
| **success** / **onSuccess** | Positive states, success messages / text on them. |
| **warning** / **onWarning** | Caution states / text on them. |
| **error** / **onError**     | Error states / text on them. |

## Interaction Layers
| Token   | Usage |
|---------|-------|
| **hover**  | Hover feedback for clickable items. |
| **active** | Pressed state feedback. |
| **focus**  | Focus ring color (keyboard accessibility). |

## Disabled
| Token              | Usage |
|--------------------|-------|
| **disabled-bg**    | Disabled background. |
| **disabled-fg**    | Disabled text/icon. |
| **disabled-border**| Disabled outline. |

## Shadows
| Token           | Usage |
|-----------------|-------|
| **shadow-high** | Stronger, tighter key shadow. |
| **shadow-low**  | Softer, wider ambient shadow. |

---

## Spacing (px)
| Token | Value | Usage |
|-------|-------|-------|
| **2xs** | 2   | Hairline spacing. |
| **xs**  | 4   | Very small gaps. |
| **sm**  | 8   | Small padding inside elements. |
| **md**  | 12  | Small gaps between elements. |
| **lg**  | 16  | Standard padding. |
| **xl**  | 24  | Section spacing. |
| **2xl** | 32  | Large gaps. |
| **3xl** | 48  | Extra-large gaps. |

## Radius (px)
| Token | Value | Usage |
|-------|-------|-------|
| **xs**  | 4   | Tiny chips, tags. |
| **sm**  | 6   | Inputs, small buttons. |
| **md**  | 10  | Cards, menus. |
| **lg**  | 14  | Large containers. |
| **xl**  | 20  | Big surfaces. |
| **pill**| 999 | Fully rounded pills/badges. |
